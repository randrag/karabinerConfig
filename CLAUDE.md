# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

Karabiner-Elements configuration for macOS keyboard remapping, using Goku to generate the config from EDN format.

## File Structure

- `karabiner.edn` - Goku source file (edit this)
- `karabiner.json` - Generated by Goku (do not edit manually)
- `automatic_backups/` - Karabiner's automatic backups

## Goku Workflow

```bash
goku          # Generate karabiner.json from karabiner.edn
gokuw         # Watch mode - regenerates on file changes
```

Karabiner auto-reloads when `karabiner.json` changes.

## Current Configuration

The existing `karabiner.json` includes:
- **Hyper key**: Caps Lock → right_shift + right_command + right_control + right_option
- **Colemak layout**: Full QWERTY→Colemak remapping
- **Hyper + home row**: Arrow keys, navigation (j/k/l/; → arrows, u/o → home/end)
- **App-specific rules**: Exceptions for Parallels, Jump Desktop, Rider
- **F# operators**: Cmd-Shift shortcuts for `|>`, `->`, `()`, `<>`
